title = '车载系统发展历史'

| 时间 | 功能（车能干什么） | 用户能力（人变强了） | 关系变化（人与车/世界） | 行业背景 / 技术触发 |
| :--- | :--- | :--- | :--- | :--- |
| **1980s–1997<br>(零代)** | 收音机、空调、基础仪表 | **基本舒适度 / 信息感知增强**<br>（知道车速/油量） | **人完全掌控 / 车只是工具**<br>人是司机，车是冷机器 | **汽车电子化萌芽**<br>微处理器开始管理引擎(ECU)，但座舱交互处于荒漠 |
| **1990s初期<br>(萌芽期)** | 早期车载 GPS 原型<br>（仅部分高端车型） | **初步路线参考**<br>（知道我在哪，但不知怎么走） | **用户仍需主动决策**<br>车提供地图，人负责规划 | **卫星导航技术实验**<br>技术极昂贵，仅作为豪华车的“电子地图展示” |
| **1998–2002<br>(第 1 代)** | 导航 / 地图显示 / 路径规划 | **不再记路线 / 车参与决策**<br>彻底消灭“迷路”的恐惧 | **空间判断权让渡给车**<br>人开始盲信系统，敢去陌生城市 | **GPS 精度解锁 (2000年)**<br>美方取消民用干扰，精度从100m→10m；城市道路复杂度激增 |
| **2005–2008<br>(第 2 代)** | 视频播放 / 蓝牙电话 /<br>倒车影像 / 后排娱乐 | **等待不再浪费 / 服务乘客**<br>车内时间价值被挖掘 | **出行体验多维分配**<br>车从服务司机转向服务全员 | **家用车普及 / 多人出行**<br>用户将家庭影院/手机通讯的预期带入车内 |
| **2009–2012<br>(第 3 代)** | 触控屏 / 统一 GUI /<br>功能可视化入口 | **操作路径缩短 / 无需学习**<br>所见即所得 | **心理门槛坍塌**<br>交互由“专业操作”变为“直觉反应” | **智能手机 (iPhone) 冲击**<br>用户交互习惯被重塑，实体按键被视为“过时” |
| **2014–2016<br>(第 4 代)** | CarPlay / Android Auto /<br>手机应用直连 | **用上好用的导航/音乐**<br>体验第一次对齐手机 | **车机承认不行 / 生态让渡**<br>车企放弃抵抗，借用手机生态救急 | **体验剪刀差 (Gap)**<br>原生车机卡顿难用，用户宁愿用手机支架 |
| **2018–2020<br>(第 5 代)** | 在线导航 / 在线音乐 /<br>语音助手 / OTA | **不带手机也能智能**<br>系统随时间变好 (常用常新) | **能力主权回归 / 独立生命体**<br>车机重新成为数据与体验的中心 | **车联网成熟 / 数据主权**<br>车企意识到不能永远把灵魂交给手机厂商 |
| **2023–至今<br>(第 6 代)** | 连续语音 / 模糊指令 /<br>主动提示 / 场景引擎 | **可说人话 / 意图被理解**<br>无需知道功能入口在哪 | **车成为人的代理 (Agent)**<br>从“人适应车”变为“车懂人” | **大模型 / 硬件同质化**<br>新能源车机械素质差异枯竭，逼迫智能化成为唯一卖点 |
| **2027–2030<br>(第 7 代·预测)** | 无感执行 / 虚实融合 (MR) /<br>生物信号交互 (眼动/脑波) | **获得“外骨骼”级感知 / 意图即现实**<br>人不再区分是自己在开还是车在帮我 | **主客体融合 (Fusion)**<br>交互消失，进入“共生”状态；归因谬误出现 | **空间计算 / 神经科学介入**<br>大模型竞争见顶，交互转向“高带宽生物连接”与混合现实 |
